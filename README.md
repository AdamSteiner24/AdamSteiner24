## Capstone Project: DnDroids

**DnDroids** is my capstone project, a web application built around a Dungeons & Dragons-style experience. The project includes user login functionality and is deployed online using AWS Elastic Beanstalk.

### Live Demo
[View DnDroids Website](http://dndroids-env.eba-6zpuswsp.us-east-2.elasticbeanstalk.com/login)

### Overview
This project was created as part of my capstone course to demonstrate full-stack web development, team collaboration, authentication, debugging, and deployment. DnDroids gave me experience working on a larger project that needed to be organized, tested, updated, and presented as a finished application.

### Technologies Used
- HTML, CSS, JavaScript
- Backend web application framework
- Git and GitHub
- AWS Elastic Beanstalk
- User authentication/login system

### My Contributions
- Helped develop and improve the web application
- Worked with GitHub for version control and team collaboration
- Tested and debugged parts of the application
- Helped prepare the project for deployment and public viewing
- Added current project information and screenshots to my portfolio

### What I Learned
This project helped me better understand how full-stack applications are built, deployed, and maintained. I also gained more experience working with a team, using GitHub, troubleshooting bugs, and presenting a technical project in a professional portfolio.
